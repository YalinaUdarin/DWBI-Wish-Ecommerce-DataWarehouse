- SCD Type 1 (Dim_Product) and SCD Type 2 (Dim_Merchant)
- SSAS Cube with Year→Quarter→Month hierarchy
- 5 OLAP operations demonstrated in Excel
- 4 Power BI reports published to Power BI Service
